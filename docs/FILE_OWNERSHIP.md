# File ownership

Every file in the project, the role that owns it, and how to claim a role.

## How to claim a role

1. Read `DESIGN.md` and this file end-to-end.
2. Reply in the team chat with the role you want: **Capture**, **Insights**, or **Reach & Data**.
3. Whoever asks first gets it. Fill in your name below.
4. Branch your work as `feat/<role>`.
